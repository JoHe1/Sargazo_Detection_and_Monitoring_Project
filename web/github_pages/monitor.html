Monitorización Near Real-Time

Sistema de detección automática de sargazo sobre imágenes Sentinel-2 descargadas periódicamente de la API de Copernicus.

Aplicación interactiva

La aplicación completa con mapa interactivo Folium, descarga automática de imágenes y visualización de detecciones está disponible en Streamlit Cloud.

Abrir app →

La app permite seleccionar una región geográfica (bounding box), una fecha y un modelo del registro, descargar la imagen Sentinel-2 correspondiente, aplicar la pipeline completa de inferencia y visualizar las detecciones de sargazo superpuestas sobre el mapa.

Pipeline de procesado

Descarga

SentinelDownloader
Copernicus API

Tiling

224×224 px
con metadatos CRS

Land mask

Natural Earth
ne_10m_land.shp

Inferencia

Swin Transformer
+ TTA opcional

Post-proceso

FAI + Gaussian
+ morfología

Mapa

Folium
georreferenciado

Cada patch de 224×224 píxeles preserva los metadatos geoespaciales (CRS, transform, bounds) del TIFF original para poder reconstruir la posición exacta de cada detección en el mapa tras la inferencia. El filtro FAI (Floating Algae Index = NIR − Rojo) actúa como filtro físico previo al umbral del modelo, eliminando falsos positivos en zonas donde el sargazo es espectralmente imposible.

Parámetros configurables

bbox[lon_min, lat_min, lon_max, lat_max] — región de interés
date_from / date_toRango de fechas para la descarga de Copernicus
max_cloud_coveragePorcentaje máximo de nubosidad aceptado (default: 40%)
patch_sizeTamaño de cada patch en píxeles (default: 224)
umbralUmbral de probabilidad para detección (default: 0.35)
umbral_faiUmbral del índice FAI (default: 0.005, 0 = desactivado)
sigmaSigma del suavizado gaussiano (default: 2.0)
min_pixelsTamaño mínimo de componente conectado (default: 50 px)
modeloNombre del modelo en ModelRegistry (default: swin_transformer)

El intervalo de actualización automática se configura directamente en el código de la app (web/streamlit_app/pages/1_monitor.py) para evitar consumo innecesario de tokens de la API de Copernicus durante el desarrollo.

Uso en local sin conexión

Modo sin internet — recomendado para la defensa del TFG

Con imágenes Sentinel-2 pre-descargadas en disco, la app funciona completamente sin conexión a internet.

1. Asegurarse de tener patches en datasets/raw_data/sentinel_downloads/

2. Activar el modo caché en el sidebar de la app

3. Ejecutar: streamlit run web/streamlit_app/app.py

4. Abrir el navegador en localhost:8501

El sitio estático (esta página) también funciona sin internet abriendo directamente el archivo docs/index.html en cualquier navegador, sin necesidad de servidor.

Clases detectadas

El modelo clasifica cada píxel en una de las 16 clases de MADOS. En el mapa se visualizan todas las clases excepto Marine Water (clase 7) y Non-annotated (clase 0).

Las clases de sargazo (2 y 3) se resaltan con mayor opacidad y constituyen la alerta principal del sistema.

El post-procesado morfológico elimina detecciones de menos de 50 píxeles, que corresponden a menos de 0.05 ha a resolución Sentinel-2 de 10 m/px.
